   **GPT võimaldab teha rohkem jaotusi(128) kui MBR(26). MBR-il on parem tarkvaraline kui ka riistvaraline ühilduvus seadmetega kui GPT. MBR tabelite maht on maksimaalselt 2TB mis on vahem kui GPT tabelitel.**
